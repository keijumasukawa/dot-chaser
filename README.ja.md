## 概要

ブラウザ上で動作するレトロアーケード風の 2D 迷路ゲーム。プレイヤーは矢印キーでキャラクターの操作を行い、ゴーストとの接触を回避しつつ、迷路内に配置されたすべてのドットを回収することでステージクリアを目指す。

> 🚧 現在開発中。実装の進行に合わせて本 README を更新する。

### デモ

https://dot-chaser-app.vercel.app

### 主要機能

- 迷路の探索とドット回収、すべて集めるとステージクリア
- それぞれ異なる追跡アルゴリズムを持つゴースト 4 体と、一定時間ゴーストを食べ返して形勢逆転できるパワーエサ
- ステージをクリアするごとに難易度が上がるレベル進行
- 残機制(残機が尽きるとゲームオーバー)とリスタート・一時停止のサポート
- スコア・ハイスコアは `localStorage` に保存

## アーキテクチャ

迷路のタイル判定・移動・衝突判定・スコア計算といったゲームロジックと Canvas への描画を分離し、ロジックは Canvas API に依存させない。メインループは `requestAnimationFrame` で駆動し、毎フレーム「入力反映 → 状態更新 → 描画」を行う。

```
[キーボード入力] ──→ [ゲームループ (requestAnimationFrame)]
                        ├─ update: 状態更新(迷路・移動・ゴーストAI・スコア)
                        └─ render: Canvas 描画(迷路・キャラクター・UI)
```

## セットアップ

必要環境: Node.js 24以上 / pnpm 11以上

```bash
pnpm install   # 依存関係のインストール
pnpm dev       # 開発サーバーの起動
```

開発サーバーは http://localhost:5173 で起動する。

環境変数の設定は不要。

## 開発コマンド

すべてリポジトリルートで実行する。

| コマンド          | 説明                                                 |
| ----------------- | ---------------------------------------------------- |
| `pnpm dev`        | 開発サーバーの起動                                   |
| `pnpm build`      | 型チェック + 本番用ビルド                            |
| `pnpm preview`    | ビルド結果のプレビュー                               |
| `pnpm test`       | 単体テストの実行(Vitest)                             |
| `pnpm test:watch` | 単体テストのウォッチ実行(ファイル変更時に自動再実行) |
| `pnpm lint`       | Lintの実行                                           |
| `pnpm format`     | コードフォーマット                                   |
