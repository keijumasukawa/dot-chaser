## プロジェクト概要

ブラウザで遊べるパックマン風の 2D 迷路ゲームです。矢印キーで自機を操作し、ゴーストを避けながら迷路のドットをすべて集めるとステージクリアです。

### 主要機能

- 迷路の探索とドット収集(すべて集めるとステージクリア)

- パワーエサ(一定時間ゴーストを食べ返せる)

- レベル進行(クリアごとに難易度上昇)
- 残機制・ゲームオーバー・リスタート
- 一時停止

## アーキテクチャ

ゲームロジック(迷路の判定・移動・衝突・スコア)と Canvas への描画を分離し、ロジックは Canvas に依存させない。メインループは `requestAnimationFrame` で駆動し、毎フレーム「入力反映 → 状態更新 → 描画」を行う。

```
[キーボード入力] ──→ [ゲームループ (requestAnimationFrame)]
                        ├─ update: 状態更新(迷路・移動・ゴーストAI・スコア)
                        └─ render: Canvas 描画(迷路・キャラクター・UI)
```

## セットアップ

必要環境: Node.js 24 以上 / pnpm 11 以上

```bash
pnpm install   # 依存関係のインストール
pnpm dev       # 開発サーバーの起動
```

環境変数の設定は不要です。

## 開発コマンド

すべてリポジトリルートで実行する。

| コマンド          | 説明                                     |
| ----------------- | ---------------------------------------- |
| `pnpm dev`        | 開発サーバーの起動                       |
| `pnpm build`      | 型チェック + 本番用ビルド                |
| `pnpm preview`    | ビルド結果のプレビュー                   |
| `pnpm test`       | 単体テストの一回実行(Vitest)             |
| `pnpm test:watch` | 単体テストの継続実行(ファイル変更を監視) |
| `pnpm lint`       | Lint の実行                              |
| `pnpm format`     | コードフォーマット                       |
